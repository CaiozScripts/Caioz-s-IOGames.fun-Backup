What is an IO game?

IO game is a special kind of online game. First of all — IO game is a game which requires nothing to play but modern web browser and Internet connection. Second — IO game is free-to-play, massively multiplayer online game. Somebody in the Internet once described IO game as “A game that easy to understand yet hard to master”. This is general yet very good description of the cool IO game. You can play it 5 minutes or you can play it 5 hours — in both cases you’ll get a lot of fun. That’s what make IO games really awesome!

There are different types of IO games. In fact you can find a game of any genre among all of IO games. It’s not just simple action games like famous Agar.io or Slither.io. There are platformers, 2D shooters or racing. You can find even RPG IO games or real-time strategies. Today almost all popular game types can be found among IO games!
